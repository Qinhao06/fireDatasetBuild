# 火焰展厅环境负样本数据集

## 数据集概述
本数据集是从火焰展厅环境视频中提取的负样本图片，用于YOLO目标检测模型的训练。

## 数据集详情
- **视频来源**: 火焰展厅环境.mp4
- **视频时长**: 143.80秒
- **视频帧率**: 29.90 FPS
- **总帧数**: 4300帧
- **抽帧策略**: 每3帧抽取1帧
- **生成图片数量**: 1434张
- **图片格式**: JPG
- **图片命名**: fire_hall_negative_XXXXXX.jpg

## 文件结构
```
fire_hall_negative_frames/
├── fire_hall_negative_000000.jpg    # 负样本图片
├── fire_hall_negative_000001.jpg
├── ...
├── fire_hall_negative_001433.jpg
└── labels/                          # YOLO格式标注文件目录
    ├── fire_hall_negative_000000.txt    # 空标注文件（负样本）
    ├── fire_hall_negative_000001.txt
    ├── ...
    └── fire_hall_negative_001433.txt
```

## 数据集特点
1. **负样本**: 所有图片均为负样本，即不包含目标对象
2. **YOLO格式**: 每张图片对应一个空的.txt标注文件
3. **环境多样性**: 涵盖火焰展厅环境的不同场景和光照条件
4. **高质量**: 从高清视频中提取，图片质量良好

## 使用方法
1. **训练数据**: 将此负样本数据集与正样本数据集合并使用
2. **验证数据**: 可用作验证集来测试模型的假阳性率
3. **数据增强**: 可对图片进行旋转、缩放等增强操作

## 注意事项
- 所有标注文件(.txt)均为空文件，这是负样本的正确格式
- 建议与正样本数据集按适当比例混合使用（如正负样本比例1:1或2:1）
- 可根据需要调整图片大小以适配不同的YOLO模型输入要求
